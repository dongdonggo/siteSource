Laravel操作数据库

连接到数据库

Laravel使用和操作数据库都很容易处理。 Laravel目前支持以下4个数据库-
  • MySQL
  • Postgres
  • SQLite
  • SQL Server

数据库查询可以使用原始SQL，查询生成器和ORM来触发。要了解Laravel所有的 CRUD(创建，读取，更新，删除)操作，我们这里将用简单的学生管理系统来演示说明。
